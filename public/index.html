D

DURE MVP

운영자-강사 워크플로우 목업

이번 단계 구현 범위

메인 홈, 로그인, 역할별 진입, 제출·관리 흐름을 하나의 웹 목업으로 정리했습니다.

1차 MVP의 핵심은 강사의 수업 기록 제출과 운영자의 기록 관리입니다. 이번 버전에서는 로그인한 역할에 따라 각자 필요한 화면만 보이도록 진입 구조도 함께 분리했습니다.

Must Flow

강사 입력에서 운영자 확인까지

  1. 로그인 또는 역할 선택
  2. 강사는 오늘 수업 통합 기록으로 바로 진입
  3. 운영자는 대시보드와 관리 화면으로 바로 진입
  4. 각 역할은 자기 업무 화면만 확인
  5. 제출과 관리 흐름이 역할별로 분리되어 진행

이번 목업 보강

비어 있던 PRD 항목

  • 메인 홈과 로그인 진입 화면
  • 역할별 메뉴 노출과 접근 제어
  • 강사 제출 검증과 최근 제출 이력 갱신
  • 운영자 권한 변경 액션과 기록 필터 동작
  • 확장 화면인 통계/성과 초안
